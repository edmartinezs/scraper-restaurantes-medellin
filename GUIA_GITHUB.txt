# GUÍA PASO A PASO PARA SUBIR A GITHUB

## PASO 1: Crear cuenta en GitHub (si no tienes)
1. Ve a https://github.com
2. Click en "Sign up"
3. Crea tu cuenta con tu email

## PASO 2: Crear un nuevo repositorio
1. Una vez dentro, click en el botón verde "New" o el "+" arriba a la derecha → "New repository"
2. Llena los datos:
   - Repository name: `scraper-restaurantes-monteria`
   - Description: `Herramienta automatizada para extraer datos de restaurantes desde Google Maps`
   - Public o Private: Elige PUBLIC (para portafolio)
   - NO marques "Add a README file" (ya tienes uno)
3. Click en "Create repository"

## PASO 3: Instalar Git en tu PC (si no lo tienes)

### Para Windows:
1. Descarga Git desde: https://git-scm.com/download/win
2. Instala con las opciones por defecto
3. Abre "Git Bash" (lo encuentras buscando en el menú inicio)

### Para Mac:
1. Abre Terminal
2. Escribe: `git --version`
3. Si no está instalado, te pedirá instalarlo automáticamente

## PASO 4: Configurar Git (primera vez)
Abre Git Bash (Windows) o Terminal (Mac) y escribe:

```bash
git config --global user.name "Tu Nombre"
git config --global user.email "tu_email@ejemplo.com"
```

## PASO 5: Preparar tu carpeta
1. Crea una carpeta nueva llamada `scraper-restaurantes-monteria`
2. Mueve estos archivos a esa carpeta:
   - scraper_restaurantes.py
   - README.md
   - requirements.txt
   - .gitignore

**NO incluyas:**
   - El archivo .xlsx con datos reales
   - Tu notebook original (.ipynb)

## PASO 6: Subir a GitHub

Abre Git Bash o Terminal, navega a tu carpeta:

```bash
cd ruta/a/tu/carpeta/scraper-restaurantes-monteria
```

Luego ejecuta estos comandos UNO POR UNO:

```bash
# Inicializar repositorio
git init

# Agregar todos los archivos
git add .

# Hacer commit
git commit -m "Primer commit: scraper de restaurantes funcional"

# Conectar con GitHub (reemplaza TU_USUARIO con tu usuario de GitHub)
git remote add origin https://github.com/TU_USUARIO/scraper-restaurantes-monteria.git

# Subir archivos
git push -u origin main
```

Si da error con "main", prueba con "master":
```bash
git branch -M main
git push -u origin main
```

## PASO 7: Verificar
1. Ve a tu repositorio en GitHub: `https://github.com/TU_USUARIO/scraper-restaurantes-monteria`
2. Deberías ver todos tus archivos

## PASO 8: Personalizar tu repo

### Agregar topics (etiquetas):
1. En tu repo, click en el engranaje junto a "About"
2. Agrega estos topics:
   - python
   - web-scraping

### Pin el proyecto en tu perfil:
1. Ve a tu perfil: `https://github.com/TU_USUARIO`

3. Selecciona este proyecto

Link: `https://github.com/TU_USUARIO/scraper-restaurantes-monteria`

## TROUBLESHOOTING

**Error: "Permission denied"**
- Configura SSH o usa HTTPS con tu token de GitHub

**Error: "Repository not found"**
- Verifica que copiaste bien la URL del repo
- Asegúrate de reemplazar TU_USUARIO con tu usuario real

**No puedo hacer push**
- Puede que necesites autenticarte con token personal
- Ve a GitHub → Settings → Developer settings → Personal access tokens
- Genera un token y úsalo como contraseña
